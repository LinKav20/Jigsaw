Краткая инструкция по использованию:
1) Запускается сервер. Вводятся данные (в программе есть инструкция). После ввода всех данных выводится сообщение об ожидании подключения, если игра не одиночная. 
2) Запускается клиент. Подключается к порту, введенному в сервере.
3) Если все хорошо, то запускаются две игры.
4) Закончить игру можно, нажав finish. Если нажать "крестик", то это принудительный выход из игры.
5) Дождаться пока второй игрок закончит и увидеть результаты.
6) Можно начать новую игру. Но. Все идет хорошо до запуска самой игры (т.е. ввод настроек работает замечатльно), а потом начинаются проблемы. Launch() можно запускать в Java только один раз. Починить не успела. За то время, что было, нормальное красивое решение не нашла( Буду рада подсказке!
